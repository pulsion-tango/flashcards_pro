# Flashcards_Pro

Flashcards_Pro is a revision app based on digital flashcards.  
It allows users to create, organize, and review content in a structured way to support long-term memory retention.

---

##  Features

- Create custom flashcards (question / answer format)
- Organize cards by topics or decks
- Review content efficiently for long-term learning

---

##  Goal

Help users improve memorization through active recall and spaced repetition principles.
